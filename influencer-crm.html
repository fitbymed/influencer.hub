Tryb lokalny
📋 Jak uruchomić chmurę (jednorazowo, ~5 min)
  1. Wejdź na supabase.com → załóż darmowe konto → New project (zapamiętaj hasło do bazy).
  2. W projekcie otwórz SQL Editor → wklej poniższy kod → Run.
  3. Wejdź w Project Settings → API. Skopiuj Project URL oraz klucz anon public.
  4. Wklej oba poniżej i kliknij Połącz. Gotowe — wpisz ten sam URL i klucz na komputerze żony.
To narzędzie wewnętrzne — nie publikuj tego pliku w internecie z wpisanym kluczem. Trzymaj go u siebie / na prywatnym dysku. Gdy będziecie chcieli, dorobimy logowanie hasłem.
